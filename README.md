**1)	Tailor Shop POS & Management System (Freelance)
**Currently deployed and used in a real tailoring shop for daily sales and inventory operations.

Live System: https://admin.markmattclothing.com


           Technologies:

.NET 8, Blazor WebAssembly, ASP.NET Core Web API, Entity Framework Core, SQL Server, Clean Architecture, CQRS
•	Designed and developed a complete Point of Sale (POS) and shop management system for a clothing retail and tailoring business.

            Key Features
•	Built POS system supporting barcode scanning and invoice generation.
•	Implemented customer management module with phone-based search.
•	Developed invoice system with automatic tax calculation.
•	Integrated thermal printer (80mm) receipt printing.
•	Created tailoring services module supporting multiple services and quantities.
•	Implemented employee shift management system.
•	Built role-based authentication (Admin / Cashier).
•	Designed dynamic receipt printing policies based on brand configuration.
•	Implemented product inventory with variants (size, color) and barcode tracking.
•	Technical Highlights
•	Applied Clean Architecture to separate Domain, Application, Infrastructure, and API layers.
•	Implemented CQRS pattern for command and query separation.
•	Developed REST APIs to support POS operations.
•	Built a responsive POS interface optimized for fast cashier workflow.
